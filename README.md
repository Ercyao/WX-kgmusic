# WX-kgmusic
基于酷狗音乐接口做的音乐播放器微信小程序

# 目录结构
* page
    *  index      首页页面
    *  ranklist   排行榜列表页面
    *  songmenu   歌单页面
    *  songlist   歌单列表页面
    *  play       播放页面
    *  profile    我的页面
    *  playlist   收藏歌曲列表页面
* img   图片和屏幕截图
* unit
    *  unit
    *  music  封装的音乐接口
