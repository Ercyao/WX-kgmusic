# WX-kgmusic
基于酷狗音乐接口做的音乐播放器微信小程序

# 目录结构
* page
    * index首页页面
    * ranklist排行榜列表页面
    * songmenu歌单页面
    * songlist歌单列表页面
    * play播放页面
    * profile我的页面
    * playlist收藏歌曲列表页面
* img   图片和屏幕截图
* unit
    * unit
    * music封装的音乐接口
